# «Rounding2» (Решение)

## Ответ

```
 a =  b * (a/b) + (a%b)
 7 =  3 *   2   +   1
 7 = -3 *  -2   +   1
-7 =  3 *  -2   +  -1
-7 = -3 *   2   +  -1
```

## Объяснение

При целочисленном делении результат всегда округляется по направлению к нулю.
При взятии остатка от деления должно выполняться следующее правило: `x mod y = x - (x / y) * y`.
